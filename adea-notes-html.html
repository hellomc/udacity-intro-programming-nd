Understanding Basics of Technology/Web

Types of files used on the web:

  • Plain Text
  • HTML
  • Images
  • Videos
  • Music

Web Components

HTML (HyperText Markup Language) = establishes content on web pages
HTTP (HyperText Transfer Protocol) = the web uses this as a main protocol
Servers = devices that host the files that are found on the web
Internet = world's largest computer network
Browser = software application that runs on computer to display files found on web

HTML Fundamentals

Typically, HTML elements will have an opening and closing tag with content in between the tags.
Some HTML elements will contain one or more attributes in the opening tag that indicate other characteristics of the HTML element.
