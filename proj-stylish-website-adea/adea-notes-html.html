Topic 1: Understanding Basics of Technology/Web

Date: 2017 October 4

How does the Web Work?

The user accesses the web by opening the browser, a software application that runs on computers. This application will display various files (i.e. plain text, HTML (hypertext markup language), images, videos, and music).

These files are hosted in servers. When a user types into the browser a website, such as www.bbc.com, the computer sends an HTTP (hypertext transfer protocol) to a server. Then the server locates the specific HTML document and sends it back to the user's computer.

HTML Fundamentals

Typically, HTML elements will have an opening and closing tag with content in between the tags.
Some HTML elements will contain one or more attributes in the opening tag that indicate other characteristics of the HTML element.

The HTML elements are either inline or block. Inline elements are containers that remain within the same line. Block elements create an invisible box around its inside content. Span and div tags are respectively inline and block elements. As seen in this paragraph, the HTML text is within a span tag and can be stylized differently from the rest of the text. The entire Topic 1 section is within a div tag that is stylized with a light blue background.

Topic 2: How to Create a Structured Document using HTML

Date: 2017 October 9

Find Developer Tools in Browser

In the Google Chrome browser, you can open the developer tool, by selecting View from the toolbar. On View's drop-down, select Developer and pick the Developer Tools option.

The shortcut is to right click the web page on the Chrome browser and select Inspect from the drop-down. The developer tools will open within the browser.

Document Structure Imitates Tree

HTML is organized in a Document Object Model(DOM) tree. This tree structure can be applied to multiple languages and across platforms to represent and interact with objects in HTML (and other markup languages).

Indenting in HTML indicates the persistence of a single idea. At the beginning of an idea, the indentation increases and towards the end the indentation decreases.

Topic 3: Cascading Style Sheets(CSS)

Date: 2017 October 12

Adding Style

CSS describes the look and formatting of HTML. It can add color and additional style.

Follows Order

CSS is applied in the following order, and as you go further down the list the CSS will override the previous styles.

  1. Browser's default style
  2. Separate file's stylesheet
  3. HTML contains stylesheet
  4. HTML element has inline style

A Cascading Waterfall

Styles apply from parent element to child elements "cascading" like a waterfall.

Selectors

CSS selectors determine which elements the CSS rules will apply to. It can be an HTML tag such as header, section, div, p, h1, etc. It can also be a class attribute that would be denoted with a period(.) or an id attribute denoted with a hashtag(#).
